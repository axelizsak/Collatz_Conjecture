## Introduction ğŸ“–

The Collatz sequence, also known as the 3n+1 sequence, is a series of numbers generated by repeatedly applying the Collatz rule. Starting with a positive integer, if the number is even, divide it by 2; if it's odd, multiply it by 3 and add 1. The sequence continues until it reaches 1. ğŸ”„

This program allows you to generate the Collatz sequence for a given positive integer and performs various checks on the sequence to validate specific criteria. âœ…

## Features âš¡ï¸

- Generates the Collatz sequence for a given positive integer.
- Performs the following checks on the generated sequence:
  - **Criterion 1** (ğŸ”): Verifies that the difference between the first binary value and its decimal equivalent is zero.
  - **Criterion 2** (ğŸ”): Checks if the difference between the last binary value and one is zero.
  - **Criterion 3** (ğŸ”): Validates that all values in the sequence are valid binary numbers (containing only '0' and '1').
  - **Criterion 4** (ğŸ”): Verifies the difference between binary values converted to decimal based on whether they are even or odd.
