## Introduction üìñ

The Collatz sequence, also known as the 3n+1 sequence, is a series of numbers generated by repeatedly applying the Collatz rule. Starting with a positive integer, if the number is even, divide it by 2; if it's odd, multiply it by 3 and add 1. The sequence continues until it reaches 1. (4, 2, 1, 4, 2, 1...) üîÑ

This program allows you to generate the Collatz sequence for a given positive integer and performs various checks on the sequence to validate specific criteria. ‚úÖ 
Polynomial constraints allow us to obtain information on certain sequences, for example by searching for recurring patterns. More widely they are used in ZKP to express the relations between the secret data of the prover and the public data known by the verifier. It is therefore essential to master this aspect of optimization on (apparently) simple problems such as Collatz. üß†
The heart of my approach is here, it is up to you to find other criteria and play with them. 
For my French speakers have a look üëÄ: https://www.youtube.com/watch?v=BP2G28694z8.

## Features ‚ö°Ô∏è

- Generates the Collatz sequence for a given positive integer.
- Performs the following checks on the generated sequence:
  - **Criterion 1** (üîç): Verifies that the difference between the first binary value and its decimal equivalent is zero.
  - **Criterion 2** (üîç): Checks if the difference between the last binary value and 1 is zero.
  - **Criterion 3** (üîç): Validates that all values in the sequence are valid binary numbers (containing only '0' and '1').
  - **Criterion 4** (üîç): Check the connection between the numbers two by two.
- Provides informative error messages for invalid input or conversion failures.

## Usage ‚öôÔ∏è

For Rust ü¶Ä:

1. Navigate to the project directory.
2. Build and run the program using the Cargo command:

   ```shell
   cargo run
   ```

3. Enter a positive integer when prompted.
4. The program will display the generated Collatz sequence and perform the checks on the sequence.



For C ü¶ç:
1. Navigate to the project directory.
2. Use the command:
  
  ```shell
   make
  ```
3. To get a trace use:
  
  ```shell
  ./collatz_proof 'any number > 0'
  ```
4. Copy the output of the terminal.
5. Type the command and enter the trace obtained:
  
  ```shell
   ./collatz_verif 'ctrl+shitf+v'
  ```
